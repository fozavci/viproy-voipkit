# SIP Modules Usage

## Register 

use auxiliary/voip/viproy_sip_register <br>
set RHOSTS 192.168.1.222 <br>
set USERNAME 201<br>
set FROM 201<br>
set PASSWORD password123<br>
set PROTO TCP<br>
set LOGIN true<br>
set DEBUG true<br>
set VERBOSE true <br>
run<br>

http://www.viproy.com/captures/sip_register.pcapng<br>
